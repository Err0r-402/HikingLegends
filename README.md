## Features
- Step tracking
- In-game shop with upgradeable multipliers
- Clean UI

## How to play
1. Walk to earn **1 coin per step**.
2. Spend coins on rank upgrades to increase your coin multiplier.
4. Keep walking and become a Hiking Legend!

## Technologies Used
- VS Code
- Garmin Connect IQ SDK
- Monkey C by Garmin
